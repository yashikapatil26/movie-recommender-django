# AI-Powered Movie Recommender 🎬
A Full-Stack web application built with **Django** and **Machine Learning**.

## 🚀 Features
- **Content-Based Filtering**: Recommends movies based on plot similarity.
- **Natural Language Processing**: Uses TF-IDF and Cosine Similarity.
- **Responsive Design**: Built with Bootstrap 5.

## 🛠️ Tech Stack
- **Backend**: Python / Django
- **Math/AI**: Pandas, Scikit-Learn
- **Version Control**: Git / GitHub

## 📝 How to Run
1. `pip install -r requirements.txt`
2. `python manage.py runserver`
